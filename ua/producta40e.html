Облік без сканера штрих-кодів

  • Ціна:$60
  • Артикул:#1201
  • Інформація для замовлення:

Повноцінна програма для обліку без використання сканера штрих-кодів. Каса працює на будь-якому пристрої з повноцінним браузером (планшет, мобільний). Для роботи системи потрібнє інтернет-з'єднання. Передбачається наявність планшета або мобільного телефону під управлінням IOS або Android для використання хмарної каси, і ноутбука або комп'ютера (один комп'ютер може бути використаний для віддаленого управління декількома торговельними точками) під керуванням ОС Windows для встановлення модуля управління складом. Є можливість використання онлайн-статистики. Так само, програмний комплекс може бути використаний на одному ноутбуці / комп'ютері, що знаходиться в торговій точці (локальний варіант).
