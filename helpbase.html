АРМ Торговля V 2.0.1

Универсальное программное обеспечение для автоматизации розничной торговли. Автоматизация магазинов, супермаркетов, кафе, ресторанов. Сделано в Украине.

На этой странице собрана информация о программе АРМ "Розничная торговля", включая ее детальное описание, видеоуроки и другой обучающий материал.

Инструкция

Инструкция к программе, детальное описание всех функций.

Базовый функционал АРМ

Базовые функции выполняемые программным обеспечением, полностью или частично встречаются во всех подобных программах.

Оборудование

  • Работа со сканером штрих-кода (любой тип подключения).
  • Работа с принтером штрих-кода, генерация собственных ш/к.
  • Работа с чекопечатью (все принтера, имеющие Windows драйвер).
  • Работа с фискальными регистраторами (Datecs, Exelio, Мария), возможность смешанного учета, работа с несколькими печатающими устройствами одновременно.
  • Поддержка считывателей дисконтных карт, печать ценников на обычном принтере, работа с дисплеем покупателя.
  • Работа как с весами с печатью этикеток, так и с подключаемыми к ПК. Прямая (из программы) загрузка весов.
  • Поддержка POS клавиатур, сенсорных дисплеев и пр.
  • Работа с терминалами сбора данных.
  • Безналичный расчет, работа с POS терминалами.

Основные функции

  • Полная номенклатура товара, категории. Информация о товаре, неограниченное кол-во полей. Себестоимость, наценка, дата прихода, срок годности, минимальный остаток и т.д.
  • Работа с накладными. Приходные, расходные, возвратные накладные. Акты переоценки (автоматические, ручные). Отсроченная переоценка.
  • FIFO метод расчета - фиксация себестоимости каждого товара. срок годности и т.д.
  • Инвентаризация: с ТСД и без, инвентаризация без закрытия, анализ итогов, локальная инвентаризация.
  • Контроль кассы, взаимозачеты с поставщиками, прочие расходы, зарплата. Расчет бонусов работникам, прибыль продавцов.
  • Дисконтные программы. Карточки, накопительные скидки, сезонные и временные скидки на товары и категории. Продажа в долг.
  • Градация доступа, неограниченное количество пользователей
  • Импорт-экспорт данных во все популярные форматы (xls, csv, html, mdb), возможность интеграции с другими программными продуктами.
  • Работа с филиалами, обмен накладными, центральный склад, работа с несколькими торговыми точками с одного рабочего места.

Отчеты и аналитика

  • Отчеты по продажам за любой период. Отчеты в разрезе чеков, товаров, применения дисконтов, операций и пр.
  • Расширенные отчеты по движению товара или группы товаров.
  • Отчеты по лидерам/аутсайдерам продаж, сезонным колебаниям спроса и т.д.
  • Возможность просмотра склада на любую дату.
  • Расширенные отчеты по действиям пользователей. Привязка всех операций в программе к пользователю.
  • Возможность создания дополнительных отчетов без привлечения производителя ПО.

Уникальный функционал АРМ

Функции, которые не реализваны в других программных продуктах

Для пользователя

  • Автоматическое подключение к базе товаров ean13.info (более 10 млн. товарных позиций) позволяет в разы сократить время на ввод номенклатуры. Время на открытие небольшого магазина (5000 наименований) в "4 руки" сокращается с 5 до 2 рабочих дней. (при сканировании кода товара его наименование автоматически "подтягивается" из глобальной базы)
  • Простота освоения, большое количество обучающего видео. Кассовые модули рассчитаны на человека, умеющего пользоваться калькулятором.
  • Надежность - программное обеспечение продолжает работать даже при отключении сети.
  • Прозрачность отчетов - все отчеты ведутся с привязкой к пользователю и имеют данные было-стало.
  • Инвентаризация с помощью мобильного телефона или планшета. Используйте HTML клиент, для облегчения инвентаризации (при отсутствии ТСД)
  • Удаленная печать чеков на сервере, при вводе данных с android/ios устройств. При этом, драйвера принтера для мобильных устройств не требуются.

Для владельца бизнеса

  • Работа программы с MySQL базой данных позволяет хранить ваши данные где угодно - на локальном компьютере, в сети, на удаленном сервере в любой стране мира. Вы защищены от несанкционированного доступа к Вашим данным, даже в случае изъятия оборудования.
  • Front End (кассовый модуль, модуль продажи) программы способен работать на любом устройстве (планшет, мобильный телефон, компьютер). Удобно для выездной торговли, кафе, точек продажи с ограниченным пространством, торговых представителей. Удаленная печать чеков на любом принтере, локальная сеть для устройств на разных платформах.
  • Удаленный доступ к отчетам позволяет контролировать бизнес из любой точки мира, с любого устройства, без установки ПО, при любой скорости интернет
  • В базовой версии ПО требуется только 1 компьютер MS Windows для Back End. Никакое другое платное стороннее программное обеспечение не требуется. Все модули успешно работают на старых компьютерах, в том числе под управлением Windows XP.
  • Возможность интеграции с интернет магазином на любой платформе - импорт и экспорт данных в один клик, синхронизация остатков.
  • Работа с несколькими магазинами с одного компьютера, подключение установленного Back End к разным базам без переустановки и смены параметров возволяет обслуживать несколько магазинов с одного рабочего места.

Для разработчика

  • Открытая модель базы данных. Используйте базу программы для создания новых модулей, отчетов, клиентов.
  • Простое создание и доработка мультиплатформенных модулей к программе. Для доработки клиентов на HTML/PHP не требуется специальных знаний. Изменения могут быть внесены даже пользователем.
  • Нет привязки к операционной системе, устройству или языку программирования. Клиент для MySQL может быть разработан для любой платформы и на любом языке программирования. Разрабатывайте свои модули, отчеты и клиенты.
