Basic functionality of AWP

The basic functions performed by the software are fully or partially found in all such programs.

Equipment

  • Work with a barcode scanner (any type of connection).
  • Work with a barcode printer, generating your own sh / c.
  • Work with check printing (all printers with a Windows driver).
  • Work with fiscal registrars (Datecs, Exelio, Maria), the possibility of mixed accounting, work with several printing devices at the same time.
  • Support for discount card readers, printing price tags on a regular printer, working with the customer’s display.
  • Work with scales with label printing, as well as connected to a PC. Direct (from the program) loading of scales.
  • Support for POS keyboards, touch displays, etc.
  • Work with data collection terminals.
  • Bank transfer, work with POS terminals.

Main functions

  • Full product range, categories. Product information, unlimited number of fields. Cost, mark-up, date of arrival, expiration date, minimum balance, etc.
  • Work with invoices. Income, expendables, return invoices. Revaluation acts (automatic, manual). Deferred revaluation.
  • FIFO calculation method - fixing the cost of each product. expiration date, etc.
  • Inventory: with TSD and without, inventory without closing, analysis of totals, local inventory.
  • Cash desk control, offsets with suppliers, other expenses, salary. Calculation of bonuses to employees, profit of sellers.
  • Discount programs. Cards, cumulative discounts, seasonal and temporary discounts on goods and categories. Selling on credit.
  • Access gradation, unlimited users
  • Import-export of data in all popular formats (xls, csv, html, mdb), the ability to integrate with other software products.
  • Work with branches, exchange of invoices, central warehouse, work with several retail outlets from one workplace.

Reports and analytics

  • Sales reports for any period. Reports broken down by checks, goods, application of discounts, transactions, etc.
  • Advanced reports on the movement of goods or groups of goods.
  • Reports on leaders / outsiders of sales, seasonal fluctuations in demand, etc.
  • The ability to view the warehouse at any date.
  • Advanced reports on user actions. Binding all operations in the program to the user.
  • Ability to create additional reports without involving a software manufacturer.

Functions that are not implemented in other software products

For the user

  • Автоматическое подключение к базе товаров ean13.info (более 10 млн. товарных позиций) позволяет в разы сократить время на ввод номенклатуры. Время на открытие небольшого магазина (5000 наименований) в "4 руки" сокращается с 5 до 2 рабочих дней. (при сканировании кода товара его наименование автоматически "подтягивается" из глобальной базы)
  • Простота освоения, большое количество обучающего видео. Кассовые модули рассчитаны на человека, умеющего пользоваться калькулятором.
  • Надежность - программное обеспечение продолжает работать даже при отключении сети.
  • Прозрачность отчетов - все отчеты ведутся с привязкой к пользователю и имеют данные было-стало.
  • Инвентаризация с помощью мобильного телефона или планшета. Используйте HTML клиент, для облегчения инвентаризации (при отсутствии ТСД)
  • Удаленная печать чеков на сервере, при вводе данных с android/ios устройств. При этом, драйвера принтера для мобильных устройств не требуются.

Для владельца бизнеса

  • Работа программы с MySQL базой данных позволяет хранить ваши данные где угодно - на локальном компьютере, в сети, на удаленном сервере в любой стране мира. Вы защищены от несанкционированного доступа к Вашим данным, даже в случае изъятия оборудования.
  • Front End (кассовый модуль, модуль продажи) программы способен работать на любом устройстве (планшет, мобильный телефон, компьютер). Удобно для выездной торговли, кафе, точек продажи с ограниченным пространством, торговых представителей. Удаленная печать чеков на любом принтере, локальная сеть для устройств на разных платформах.
  • Удаленный доступ к отчетам позволяет контролировать бизнес из любой точки мира, с любого устройства, без установки ПО, при любой скорости интернет
  • В базовой версии ПО требуется только 1 компьютер MS Windows для Back End. Никакое другое платное стороннее программное обеспечение не требуется. Все модули успешно работают на старых компьютерах, в том числе под управлением Windows XP.
  • Возможность интеграции с интернет магазином на любой платформе - импорт и экспорт данных в один клик, синхронизация остатков.
  • Работа с несколькими магазинами с одного компьютера, подключение установленного Back End к разным базам без переустановки и смены параметров возволяет обслуживать несколько магазинов с одного рабочего места.

Для разработчика

  • Открытая модель базы данных. Используйте базу программы для создания новых модулей, отчетов, клиентов.
  • Простое создание и доработка мультиплатформенных модулей к программе. Для доработки клиентов на HTML/PHP не требуется специальных знаний. Изменения могут быть внесены даже пользователем.
  • Нет привязки к операционной системе, устройству или языку программирования. Клиент для MySQL может быть разработан для любой платформы и на любом языке программирования. Разрабатывайте свои модули, отчеты и клиенты.
